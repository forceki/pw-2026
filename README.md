# Praktikum Website 2026 - Universitas Pasundan

Repository ini digunakan sebagai **materi perkuliahan Praktikum Website tahun 2026** di **Universitas Pasundan (UNPAS)**.

Repo ini berisi kumpulan latihan, contoh kode, dan proyek praktikum yang digunakan selama perkuliahan. Setiap folder mewakili pertemuan atau topik praktikum yang berbeda.

---

## Struktur Repository

* **pertemuan1/** — Materi dan latihan pertemuan 1
* **pertemuan2/** — Materi dan latihan pertemuan 2
* **perpustakan/** - Web Praktikum Management Perpustakaan
* *(folder berikutnya akan ditambahkan sesuai perkembangan materi)*

---

## Tujuan Repository

Repository ini dibuat untuk membantu mahasiswa:

* Memahami dasar pengembangan website
* Belajar menggunakan framework Laravel
* Mempraktikkan konsep pemrograman web secara langsung
* Mengelola proyek menggunakan Git dan GitHub

---

## Cara Menggunakan

1. Clone repository ini:

   ```bash
   git clone <repository-url>
   ```

2. Masuk ke folder pertemuan yang ingin dipelajari.

3. Ikuti instruksi yang ada pada masing-masing folder latihan.

---

## Catatan

Repository ini akan terus diperbarui selama semester berlangsung sesuai dengan materi perkuliahan.
